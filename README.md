# JardinSolidaire
Projet RNCP

Install :

- Node.js
- Docker Desktop


Docker :

- Create Docker Project :

mkdir JardinSolidaire
cd JardinSolidaire
touch docker-compose.yml

- Build Docker Project :

docker-compose up --build

- Stop Docker Project

docker-compose down


React App :

- Create React App :

npm create vite@latest my-react-app --template react

- Install :

npm install

- Run :

npm run dev
